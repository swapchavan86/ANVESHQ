# Project Overview

This is a Python backend project.

## Tech Stack
- Language: Python
- Database: SQLite
- Email reporting enabled
- Finance data via Yahoo Finance

## Folder Structure
- Backend/src : core application logic
- Backend/data : sqlite databases
- Backend/templates : HTML templates
- Backend/tests : unit tests

## Entry Point
- main.py

## Key Files
- database.py : DB connection & queries
- services.py : business logic
- cleanup_service.py : background cleanup
- email_report.py : email generation
- yahoo_finance.py : finance data fetch
